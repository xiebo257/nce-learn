## EN: 1. Cave exploration, or pot-holing, as it has come to be known, is a relatively new sport.
美音发音: /keɪv ˌɛkspləˈreɪʃən ɔr ˈpɑtˌhoʊlɪŋ æz‿ɪt hæz kʌm tə bi noʊn ɪz‿ə ˈrɛlətɪvli nu spɔrt/

## 词汇总结
洞穴地貌: cave 洞穴； chasm 裂谷； plateau 高原； corridor 通道； waterfall 瀑布； underground lake 地下湖； cavern 大洞穴； cleft 裂缝； dome 穹顶；
探险动作: explore 探索； descent 下探； rig up 架设； edge one's way 缓慢移动； wade 涉水； plunge into 跳入； squeeze through 挤过； switch on 打开；

## 句型总结
`it is...that...`: 强调句，用于突出吸引人们下洞的原因。
`It is impossible to do...`: It 作形式主语，不定式是真正主语。
`had not the entrance been spotted`: 省略 if 的虚拟条件倒装。
`remains to be explored`: remain + to be done 表“仍有待……”。
`saw...rising up...`: see + 宾语 + 现在分词，强调正在呈现的景象。

## 时态总结
一般现在时用于说明洞穴探险的普遍事实：`is`, `have`, `require`, `are necessary`。
现在完成时用于洞穴形成、命名和发现后的影响：`has come to be known`, `has been formed`, `has become`。
一般过去时用于具体探险过程：`went down`, `climbed`, `came`, `plunged`, `arrived`, `saw`。
过去完成时被动用于更早发生的地貌变化：`had been washed up`。

## 情态/语气总结
`can only be dimly understood` 表能力/可能性有限。
`can take as long as...` 表可能需要长达多少时间。
`might never have been discovered` 是对过去事实相反情况的推测。
`could hear` 和 `could be heard` 分别表示过去能力和情态被动。

## 本课连读注意点
辅音 + 元音连读: `as it` -> /æz‿ɪt/；`chance of` -> /tʃæns‿əv/；`rig up` -> /rɪɡ‿ʌp/；`washed up` -> /wɑʃt‿ʌp/；`switching on` -> /ˈswɪtʃɪŋ‿ɑn/。
元音 + 元音滑读: `into a` -> /ˈɪntu‿ə/；`to a depth` -> /tə‿ə dɛpθ/；`the entrance` -> /ði ˈɛntrəns/。
弱读词: `of`, `to`, `and`, `as`, `for` 多弱读，如 `depths of the earth` -> /dɛpθs‿əv ði‿ɝθ/。
意群停顿: 第 27 句可按 `After switching on... / they saw great stalagmites / some of them... / rising up...` 分组。
