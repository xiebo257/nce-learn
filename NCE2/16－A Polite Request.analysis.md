# NCE2 Lesson 16 - A Polite Request

问题: What was the polite request?

1.
EN: 1. If you park your car in the wrong place, a traffic policeman will soon find it.
美音发音: /ɪf ju pɑrk jɚ kɑr‿ɪn ðə rɔŋ pleɪs‿ə ˈtræfɪk pəˈlismn wɪl sun find‿ɪt/
连读分析: kɑr‿ɪn、pleɪs‿ə、find‿ɪt 连读；ðə、ə、jɚ 弱读/弱化。
结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 将来结构表示预期或安排。
中: 如果你把车停错地方，交警很快就会发现。
词: park 公园； wrong 错误的； place 地方； traffic 交通；

2.
EN: 2. You will be very lucky if he lets you go without a ticket.
美音发音: /ju wɪl bi ˈvɛri ˈləki ɪf hi lɛt ju ɡoʊ wəˈðaʊt‿ə ˈtɪkət/
连读分析: wəˈðaʊt‿ə 连读；ə 弱读/弱化。
结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 将来结构表示预期或安排。
中: 如果他不带票就放你走，那你就很幸运了。
词: very 非常； lucky 幸运的； lets 让我们； without 没有；

3.
EN: 3. However, this does not always happen.
美音发音: /haʊˈɛvər ðɪs dəz nɑɾ‿ˈɔlˌweɪz ˈhæpən/
连读分析: nɑɾ‿ˈɔlˌweɪz 连读；nɑɾ‿ˈɔlˌweɪz 中 /ɾ/ 为美式闪音。
结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 一般现在时或基础事实陈述。
中: 然而，这种情况并不总是发生。
词: however 然而； this 这； always 总是； happen 发生；

4.
EN: 4. Traffic police are sometimes very polite.
美音发音: /ˈtræfɪk pəˈlis‿ɑr ˈsəmˌtaɪmz ˈvɛri pəˈlaɪt/
连读分析: pəˈlis‿ɑr 连读；ɑr 弱读/弱化。
结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 一般现在时或基础事实陈述。
中: 交警有时很有礼貌。
词: traffic 交通； police 警察； sometimes 有时； very 非常；

5.
EN: 5. During a holiday in Sweden, I found this note on my car: 'sir, we welcome you to our city.
美音发音: /ˈdʊrɪŋ‿ə ˈhɑləˌdeɪ ɪn ˈswidn aɪ found ðɪs note ɔn maɪ kɑr sər wi ˈwɛlkəm ju tə aʊɚ ˈsɪdi/
连读分析: ˈdʊrɪŋ‿ə 连读；ə、tə 弱读/弱化。
结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 一般过去时叙述故事主线。
中: 在瑞典度假期间，我在车上发现了这张纸条：“先生，我们欢迎您来到我们的城市。”
词: during 期间； holiday 假期； sweden 瑞典； found 成立；

6.
EN: 6. This is a No Parking area.
美音发音: /ðɪs‿ɪz‿ə noʊ ˈpɑrkɪŋ‿ˈɛriə/
连读分析: ðɪs‿ɪz‿ə、ˈpɑrkɪŋ‿ˈɛriə 连读；ə、ɪz 弱读/弱化。
结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 一般现在时或基础事实陈述。
中: 这是禁止停车区。
词: this 这； parking 停車處； area 区域；

7.
EN: 7. You will enjoy your stay here if you pay attention to our street signs.
美音发音: /ju wɪl‿ɪnˈdʒɔɪ jɚ steɪ hɪr‿ɪf ju peɪ‿əˈtɛnʃən tə aʊɚ strit signs/
连读分析: wɪl‿ɪnˈdʒɔɪ、hɪr‿ɪf、peɪ‿əˈtɛnʃən 连读；tə、jɚ 弱读/弱化。
结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 将来结构表示预期或安排。
中: 如果您留意我们的街道标志，您将会在这里享受愉快的住宿体验。
词: pay attention 注意； enjoy 享受； stay 停留； here 这里； attention 注意；

8.
EN: 8. This note is only a reminder.'
美音发音: /ðɪs note ɪz‿ˈoʊnli ə reminder/

时态: 一般现在时或基础事实陈述。

9.
EN: 9. If you receive a request like this, you cannot fail to obey it!
美音发音: /ɪf ju rəˈsiv‿ə rəˈkwɛst like ðɪs ju ˈkæˌnɑt feɪl tə əˈbeɪ ɪt/

结构: 主语 + 谓语 + 宾语/补足语的基础叙述句。
时态: 一般现在时或基础事实陈述。

词: receive 收到； request 请求； like 喜欢； this 这；
