# NCE1 American-Pronunciation Audio

Place licensed American-pronunciation MP3 files in this directory using the same
filenames as the existing `NCE1/*.mp3` lesson-pair audio files, for example:

```text
001&002－Excuse Me.mp3
003&004－Sorry, Sir..mp3
```

Generated NCE1 analysis pages prefer files from this directory. If a file is
missing or cannot be loaded, the player falls back to the original `NCE1/*.mp3`
file with the same name.

If the American recordings have different pacing, regenerate or retime the
matching `.lrc` files before relying on sentence-level playback.
