词汇总结

教堂与钟: vicar 牧师； church clock 教堂钟； bell 钟； clock tower 钟楼； strike the hours 报时；

动作短语: raise money 筹钱； have sth repaired 请人修理； wake up 醒来/叫醒； catch sight of 看见； get used to 习惯；

语气表达: with a start 猛然一惊； in surprise 惊讶地； all right 正常； better than nothing 总比没有强；

句型总结

be always doing 可表示反复行为并带感情色彩。

have sth done 表请别人做某事。

Armed with... 过去分词短语作伴随状语。

not as...as... 和 better than... 用于比较。

时态总结

现在完成时表示持续结果：has been silent, has never managed。

过去进行时描写夜里正在发生的事：was striking。

现在完成进行时 I've been coming... 表持续反复动作。

情态/语气总结

will strike 表将来会发生；can do about it 表处理能力。

Let's... 是建议语气；did give 强调过去动作。

本课连读注意点

注意 wake up、went up into、woken up everyone 的连读。

美式闪音: better /ˈbɛɾɚ/、at one 可轻读连起来。

直接引语要读出惊讶、解释和安慰的语气。
